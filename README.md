# rutinetrine

Funksjoner som er løst:

- Opprette bruker, enten admin eller employer
- Admin kan poste en rutine og velge en employer som skal utføre
- Employer vil få sin utdelte rutine opp i sitt dashboard
- Alle som er admin vil se alle rutinene, men det står hvem som har postet og hvem som skal utføre rutine

Funksjoner som ikke ble løst

- Logge inn med google auth. Jeg begynte med lokal login siden det har jeg gjort noen ganger før. Har vært innom google auth så tror jeg skal fint løse det, men rakk ikke det denne gangen
- Sette rutiner som fullført. Jeg har gjort noe back-end på dette, men har ikke løst dette i front-end. Om dere øsnker legger jeg gjerne til denne funksjonen, men hadde ikke tid til det nå
- Slutt-tid på rutine. Og kunne kanskje spandert på med moment.js :) 
- Automatisk oppfølging av rutine
